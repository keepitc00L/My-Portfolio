# Aarav Deshpande - Personal Portfolio 🚀

A high-performance, cinematic personal portfolio built to showcase my projects, skills, and journey as a 2nd-year engineering student. Designed with a clean, minimalist aesthetic inspired by modern web design, featuring a crisp white background, bold typography, and dynamic animations.

## ✨ Features
- **Cinematic Design**: High-contrast, typography-driven aesthetic with smooth scrolling.
- **Dynamic Background**: Interactive, flowing geometric canvas background that adds depth.
- **Custom Cursor**: Custom-built interactive dot and ring cursor for a premium feel.
- **Scroll Reveal Animations**: Elements gracefully fade and slide in as you scroll.
- **Interactive Lightbox**: Full-screen certificate & achievement viewer.
- **Fully Responsive**: Flawless experience across desktop, tablet, and mobile devices.

## 🛠️ Tech Stack
- **HTML5**: Semantic and accessible structure.
- **CSS3**: Vanilla CSS for maximum performance, featuring modern CSS variables, Grid, and Flexbox layouts.
- **JavaScript (ES6+)**: Custom vanilla JS for the loading screen, scroll observer, stat counter, canvas animations, and lightbox logic (No heavy libraries/frameworks).

## 📂 Sections
1. **Hero**: Striking introduction with a custom percentage loader.
2. **About**: Personal journey, fitness transformation stats, and an editorial-style photo collage.
3. **Skills**: Minimalist grid highlighting core technical competencies (C++, DSA, Python, JS).
4. **Projects**: Selected GitHub repositories with clean hover effects.
5. **Achievements**: Gallery of certifications and problem-solving milestones (Google, LeetCode, HackerRank).
6. **Journey**: A chronological timeline of key events and growth.
7. **Contact**: Quick links to GitHub, Instagram, and Email.
